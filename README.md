# VitalBuild

VitalBuild is a Next.js + TypeScript project that leverages **Gemini Pro 3.2** to solve complex agentic issues across domains such as **Environment, Healthcare, Risk Prevention, and more**. Depending on the input prompt, VitalBuild dynamically selects and guides the appropriate Gemini-based agent to provide actionable insights and solutions.

This project represents a **new age of AI Agentic Businesses**, combining cutting-edge generative AI with translation and multimedia capabilities.

---

## ✨ Features

- **Gemini Pro 3.2 Integration**  
  Uses the latest Gemini model for intelligent content generation and agentic reasoning.

- **Multi-Agentic Problem Solving**  
  Dynamically routes prompts to specialized agents (environment, healthcare, risk prevention, etc.).

- **Google Translate Support**  
  Automatically translates user prompts into English for better AI comprehension.

- **Google Photos Integration**  
  Allows users to upload images (base64) which are stored securely in Google Photos.

- **Enhanced Frontend UI**  
  Modern form-based interface with support for intent, prompt entry, and image uploads.

- **Secure Backend**  
  Environment variables are managed via `.env.local` and never baked into Docker images.
